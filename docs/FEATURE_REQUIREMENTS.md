This document defines _what Endure must do_, not how.

---

## Athlete

Must be able to:

- Plan training weeks
- Create sessions by day
- See planned vs completed instantly
- Sync workouts (Garmin)
- Analyze completed sessions
- Track weekly and long-term load
- Detect over/under training visually

---

## Coach

Must be able to:

- View assigned athletes
- Switch between athletes
- Create training plans
- Overlay plans onto athlete calendars
- Review athlete compliance
- Analyze athlete sessions
- (Future) Sell training plans

---

## Admin

Must be able to:

- Approve / deny coaches
- View all registered users
- View paid vs free users
- Impersonate athlete or coach
- Monitor system health (later)

---

## Subscription / Paywall

Free:

- Limited history
- Basic metrics
- Single athlete

Paid:

- Long-term analysis
- Coach features
- Plan overlays
- Advanced metrics

---

## Explicitly Out of Scope (V1)

- Messaging
- Social feed
- Nutrition
- Mobile-native UI
